                  💼 ДЛЯ HR-СПЕЦИАЛИСТОВ
═══════════════════════════════════════════════════════════════

РЕГИСТРАЦИЯ:
  1. Перейти на /register
  2. Выбрать "HR-специалист"
  3. Заполнить форму

НАСТРОЙКА ПРОФИЛЯ:
  1. Профиль → Редактировать
  2. Заполнить:
     • Название компании
     • Описание компании
     • Контакты (WhatsApp, Instagram, LinkedIn)
  3. Загрузить аватар

СОЗДАНИЕ ВАКАНСИИ:
  1. Профиль → "+ Добавить вакансию"
  2. Заполнить:
     • Название (напр. "Senior Python Developer")
     • Описание (детали вакансии)
     • Требования (опыт, знания)
     • Навыки (Python, Django, PostgreSQL)
     • Локация (Алматы / Удаленно)
     • Зарплата (500,000 - 800,000 тг)
     • Тип (Полная занятость)
  3. Сохранить

ПОИСК КАНДИДАТОВ:
  1. Навигация → "Кандидаты"
  2. Ввести: имя / навык / должность
  3. Просмотр результатов
  4. Клик на карточку → полный профиль

AI-АНАЛИЗ КАНДИДАТА:
  1. Профиль кандидата → "Анализировать с вакансией"
  2. Выбрать свою вакансию
  3. Нажать "Анализировать"
  4. Получить отчет:
     - % соответствия
     - Сильные/слабые стороны
     - Недостающие навыки
     - Вопросы для собеседования
     - Рекомендации

ОТПРАВКА ЗАПРОСА:
  1. Профиль кандидата → прокрутить вниз
  2. Заполнить сообщение
  3. Отправить запрос
  4. Кандидат увидит в уведомлениях 🔔

ЧАТ:
  1. Добавить ссылку в профиле кандидата
  2. Или: /chat/{candidate_id}
  3. Писать сообщения
  4. История сохраняется

ОТЗЫВ О КАНДИДАТЕ:
  1. После работы: /review/create
  2. Выбрать рейтинг (1-5 звезд)
  3. Написать комментарий
  4. Отзыв появится в профиле

═══════════════════════════════════════════════════════════════
                  👔 ДЛЯ КАНДИДАТОВ
═══════════════════════════════════════════════════════════════

РЕГИСТРАЦИЯ:
  1. Перейти на /register
  2. Выбрать "Кандидат"
  3. Заполнить форму

НАСТРОЙКА ПРОФИЛЯ:
  1. Профиль → Редактировать
  2. Заполнить:
     • Заголовок (Senior Developer)
     • Локация (Алматы, Казахстан)
     • О себе
     • Навыки (важно!)
     • Контакты
     • Социальные сети
  3. Загрузить аватар

ЗАГРУЗКА РЕЗЮМЕ:
  1. Профиль → "Upload Resume"
  2. Выбрать файл (PDF или DOCX, макс 10MB)
  3. Загрузить
  4. Добавить навыки (рекомендуется)

ДОБАВЛЕНИЕ ПОРТФОЛИО:
  1. Использовать API: POST /portfolio/add
  2. Поля:
     - title: Название проекта
     - description: Описание
     - project_url: Ссылка на проект
     - technologies: Python, React, PostgreSQL

ДОБАВЛЕНИЕ СЕРТИФИКАТОВ:
  1. Использовать API: POST /certificates/add
  2. Поля:
     - title: Название сертификата
     - issuer: Организация
     - issue_date: Дата выдачи
     - credential_url: Ссылка на подтверждение

УВЕДОМЛЕНИЯ:
  1. Колокольчик 🔔 в навигации
  2. Показывает количество новых запросов
  3. Клик → список запросов
  4. В каждом запросе:
     - Имя HR и компания
     - Сообщение
     - Кнопки контактов (WhatsApp, Instagram, LinkedIn)
     - Кнопка "Отменить"

АНАЛИЗ РЕЗЮМЕ:
  1. Навигация → "Анализ"
  2. Загрузить резюме (если не загружено)
  3. Вставить описание вакансии мечты
  4. Получить анализ:
     - % соответствия
     - Плюсы/минусы
     - Рекомендации

ЧАТ С HR:
  1. Ссылка из уведомления
  2. Или: /chat/{hr_id}
  3. Общение в реальном времени

ОТЗЫВ О HR:
  1. После сотрудничества: /review/create
  2. Оценить (1-5 звезд)
  3. Написать комментарий

СТРУКТУРА БД:
  • User - пользователи (HR и кандидаты)
  • Job - вакансии
  • Analysis - анализы резюме
  • Request - запросы HR к кандидатам
  • Message - сообщения в чате
  • Review - отзывы
  • Portfolio - портфолио проектов
  • Certificate - сертификаты
  • Interview - интервью (для будущего)

ОСНОВНЫЕ РОУТЫ:
  /                - главная
  /register        - регистрация
  /login           - вход
  /profile         - профиль
  /dashboard       - панель
  /candidates      - поиск кандидатов (HR)
  /candidate/{id}  - профиль кандидата
  /jobs/create     - создание вакансии
  /hr/analyze-candidate/{id} - AI-анализ
  /chat/{id}       - чат
  /analyze         - анализ резюме (кандидат)

API:
  GET  /api/notifications-count  - количество уведомлений
  GET  /api/my-requests           - список запросов
  POST /api/dismiss-request/{id}  - удалить запрос
  POST /portfolio/add             - добавить проект
  POST /certificates/add          - добавить сертификат
  POST /review/create             - создать отзыв

AI-МОДЕЛЬ:
  • Ollama: gpt-oss:20b-cloud
  • Детальный анализ на русском
  • JSON-формат ответа

ФАЙЛЫ ЗАГРУЗОК:
  uploads/avatars/       - аватары
  uploads/resumes/       - резюме
  uploads/videos/        - видео-резюме
  uploads/certificates/  - сертификаты
  uploads/portfolio/     - изображения проектов

Q: Как удалить вакансию?
A: В профиле HR → кнопка "Удалить" рядом с вакансией

Q: Как HR может анализировать кандидата?
A: Профиль кандидата → "Анализировать с вакансией"

Q: Где видны отзывы?
A: В профиле пользователя, под основной информацией

Q: Как добавить портфолио?
A: POST запрос на /portfolio/add (пока нет UI)

Q: Нужен ли Ollama?
A: Да, для AI-анализа. Запустите: ollama serve

Q: Что если AI не работает?
A: Будет fallback-анализ с базовыми данными

Q: Как сменить роль (HR ↔ Кандидат)?
A: Нельзя, нужно создать новый аккаунт

Q: Где хранятся данные?
A: В файле hr_agent.db (SQLite)
