- Статус прочтения

- Взаимные отзывы
- Рейтинг 1-5 звезд

### 7. Interview (Интервью)
- Планирование встреч
- Ссылки на видеозвонки

## 🔧 Технические детали

### API эндпоинты (будут добавлены):

**Вакансии:**
- `GET /jobs` - список вакансий
- `POST /jobs/create` - создать вакансию
- `GET /job/{id}` - просмотр вакансии
- `POST /job/{id}/edit` - редактировать
- `POST /job/{id}/delete` - удалить

**AI-анализ для HR:**
- `POST /hr/analyze-candidate` - анализ резюме с вакансией

**Портфолио:**
- `POST /portfolio/add` - добавить проект
- `GET /portfolio/{id}` - просмотр проекта
- `POST /portfolio/{id}/delete` - удалить

**Сертификаты:**
- `POST /certificates/add` - добавить сертификат
- `GET /certificates` - список сертификатов

**Чат:**
- `GET /chat/{user_id}` - открыть чат с пользователем
- `POST /chat/send` - отправить сообщение
- `GET /api/messages/unread` - непрочитанные сообщения

**Отзывы:**
- `POST /review/create` - создать отзыв
- `GET /reviews/{user_id}` - отзывы пользователя

**Интервью:**
- `POST /interview/schedule` - запланировать
- `GET /interviews` - список интервью

## 📝 Как использовать (после реализации)

### HR-специалист:

1. **Добавить компанию в профиль:**
   - Перейти в Профиль → Редактировать
   - Заполнить "Название компании" и "О компании"

2. **Создать вакансию:**
   - Профиль → "Добавить вакансию"
   - Заполнить все поля
   - Указать требуемые навыки

3. **Анализ кандидата:**
   - Профиль кандидата → "Анализировать с вакансией"
   - Выбрать свою вакансию
   - Получить AI-анализ соответствия

4. **Написать кандидату:**
   - Профиль кандидата → "Написать"
   - Открывается чат

5. **Запланировать интервью:**
   - Профиль кандидата → "Запланировать интервью"
   - Указать дату и время
   - Добавить ссылку Zoom/Meet

### Кандидат:

1. **Добавить проект в портфолио:**
   - Профиль → "Портфолио" → "Добавить проект"
   - Название, описание, ссылка, технологии

2. **Загрузить видео-резюме:**
   - Профиль → Редактировать → "Видео-резюме"
   - Загрузить MP4 (макс 50MB)

3. **Добавить сертификат:**
   - Профиль → "Сертификаты" → "Добавить"
   - Название, организация, ссылка на проверку

4. **Просмотр вакансий:**
   - Навигация → "Вакансии"
   - Фильтр по навыкам/локации
   - "Откликнуться" на вакансию

5. **Чат с HR:**
   - Уведомления → "Написать" HR
   - Обмен сообщениями
