🎯 ЧТО ЭТО?

   AI-платформа, которая создает полноценные MVP из ваших идей:
   
   ✨ Пишете идею (даже неполную)
   ✨ AI дополняет детали сам
   ✨ Генерирует HTML + Tailwind + React
   ✨ Показывает live preview результата
   ✨ Объясняет через markdown

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🚀 БЫСТРЫЙ СТАРТ:

   1. Запустите:
      python project0.py
   
   2. Откройте:
      http://localhost:8000
   
   3. Опишите идею:
      "Создай landing page для стартапа"
   
   4. Нажмите ⚡ (Generate)
   
   5. Смотрите результат в live preview!

🛠️ УСТАНОВКА OLLAMA:

   1. Скачайте: https://ollama.ai/
   
   2. Установите Ollama
   
   3. Запустите сервер:
      ollama serve
   
   4. Установите модель:
      ollama pull glm-4.6:cloud
   
   5. Готово!
