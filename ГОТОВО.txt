🚀 БЫСТРЫЙ ЗАПУСК:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1️⃣ Убедитесь что Ollama запущен:
   ollama serve

2️⃣ Загрузите модель (если не загружена):
   ollama pull gpt-oss:20b-cloud

3️⃣ Запустите приложение:
   
   Способ А (скрипт):
   ./ЗАПУСК.sh
   
   Способ Б (uvicorn):
   uvicorn hr:app --host 0.0.0.0 --port 8000 --reload
   
   Способ В (python):
   python3 hr.py

4️⃣ Откройте браузер:
   http://localhost:8000

✅ ЧТО ОСТАВЛЕНО (для кандидатов):

✓ Регистрация и вход
✓ LinkedIn-style профиль:
  - Аватар (фото профиля)
  - Полное имя, заголовок, локация
  - О себе (bio)
  - Телефон
  - Навыки (для точного AI-анализа)
  - Социальные сети (LinkedIn, GitHub, сайт)
  - Загрузка резюме (PDF/DOCX)

✓ AI-анализ резюме:
  - Загрузка PDF или DOCX
  - Описание вакансии
  - Использование навыков из профиля
  - Детальный анализ (7-10 пунктов):
    → % соответствия (0-100)
    → 7-10 сильных сторон с примерами
    → 7-10 слабых сторон с путями решения
    → Навыки: есть/не хватает/дополнительные
    → Анализ опыта (с оценкой %)
    → Анализ образования (с оценкой %)
    → 7-10 практичных рекомендаций
    → Итоговое резюме (3-5 предложений)

✓ История анализов:
  - Все прошлые результаты
  - Статистика (количество, средний балл)
  - Быстрый доступ к результатам

✓ Панель управления:
  - Обзор активности
  - Последние анализы
  - Быстрый доступ к функциям

💡 СОВЕТЫ ПО ИСПОЛЬЗОВАНИЮ:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Заполните профиль полностью:
   • Добавьте фото
   • Загрузите резюме
   • Укажите РЕАЛЬНЫЕ навыки
   • Заполните "О себе"

2. Указывайте навыки точно:
   • Только те, которыми владеете
   • Через запятую или с новой строки
   • Это критично для точного анализа!

3. При анализе:
   • Используйте галочку "Использовать навыки из профиля"
   • Вставляйте ПОЛНОЕ описание вакансии
   • Чем больше деталей, тем точнее анализ

4. Анализируйте разные вакансии:
   • Сравнивайте результаты
   • Работайте над слабыми сторонами
   • Следуйте рекомендациям AI
