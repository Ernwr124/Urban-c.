🚀 КАК ИСПОЛЬЗОВАТЬ:

⚠️  ВАЖНО: ОБЯЗАТЕЛЬНО ПЕРЕСОЗДАЙТЕ БД!

1. Удалите старую базу:
   rm hr_agent.db

2. Запустите платформу:
   python3 hr_platform.py

3. Откройте браузер:
   http://localhost:8000

4. Зарегистрируйтесь:
   • Как HR-специалист - для найма
   • Как кандидат - для поиска работы

══════════════════════════════════════════════════════════════

💼 ДЛЯ HR-СПЕЦИАЛИСТОВ:

1. ✅ Заполните профиль:
   - Название компании
   - Описание компании
   - Контакты (WhatsApp, Instagram)

2. ✅ Создайте вакансию:
   - Профиль → "+ Добавить вакансию"
   - Заполните все поля
   - Сохраните

3. ✅ Ищите кандидатов:
   - Навигация → "Кандидаты"
   - Поиск по имени/навыкам
   - Просмотр профилей

4. ✅ Анализируйте кандидатов:
   - Профиль кандидата → "Анализировать с вакансией"
   - Выберите свою вакансию
   - Получите AI-отчет

5. ✅ Связывайтесь:
   - Отправьте запрос
   - Напишите в чат
   - Оставьте отзыв после работы

══════════════════════════════════════════════════════════════

👔 ДЛЯ КАНДИДАТОВ:

1. ✅ Заполните профиль:
   - Загрузите резюме
   - Добавьте навыки
   - Загрузите аватар

2. ✅ Добавьте портфолио:
   - Профиль → "Портфолио"
   - Добавьте проекты
   - Ссылки на работы

3. ✅ Добавьте сертификаты:
   - Профиль → "Сертификаты"
   - Название, организация
   - Ссылка на подтверждение

4. ✅ Получайте запросы:
   - Колокольчик 🔔 в навигации
   - Просмотр запросов от HR
   - Контакты HR для связи

5. ✅ Общайтесь:
   - Чат с HR
   - Оставляйте отзывы

🔧 ТЕХНИЧЕСКИЕ ДЕТАЛИ:

НОВЫЕ РОУТЫ:

HR:
  GET  /jobs/create - создание вакансии
  POST /jobs/create - сохранение вакансии
  POST /job/{id}/delete - удаление вакансии
  GET  /hr/analyze-candidate/{id} - страница анализа
  POST /hr/analyze-candidate - выполнение анализа

ЧАТ:
  GET  /chat/{user_id} - страница чата
  POST /chat/send - отправка сообщения

РЕЙТИНГИ:
  POST /review/create - создание отзыва

ПОРТФОЛИО:
  POST /portfolio/add - добавление проекта
  POST /certificates/add - добавление сертификата

ОБНОВЛЕННЫЕ РОУТЫ:
  POST /update-profile - теперь поддерживает company_*
